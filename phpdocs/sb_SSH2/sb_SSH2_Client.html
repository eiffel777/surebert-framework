Class sb_SSH2_Client

Description
Description | Descendents | Vars (details) | Methods (details)

Create an SSH2 client for executing commands also the basis of SFTP client requires php-pecl-ssh2 extension see http://php.net/manual/en/wrappers.ssh2.php for more info

  1.  $client = new sb_SSH2_Client('server.com', 1027);
  2.  $client->login('uname', 'pass');
  3.  OR
  4.  $client->login_with_key('uname', 'id_rsa.pub', 'id_rsa', '');
  5.  
  6.  $data = $client->exec('ls /tmp');
