Class sb_SFTP_Client

Description
Description | Vars (details) | Methods (details)

Implements interfaces:

  • sb_FTP_Base

Uses the pecl ssh2 extension in order based on code sample from php manual http://www.php.net/manual/en/ref.ssh2.php requires php-pecl-ssh2 extension

  1.  try{
  2.         $client = new sb_SFTP_Client("server.com", 1027);
  3.         $client->login("uname", "pass");
  4.         OR
  5.         $client->login_with_key('uname', 'id_rsa.pub', 'id_rsa', '');
  6.         $client->put("/tmp/to_be_sent", "/tmp/to_be_received");
  7.         $client->get("/tmp/to_be_received", "/tmp/to_be_sent_new");
  8.  } catch (Exception $e){
  9.         echo $e->getMessage() . "\n";
  10.  }
