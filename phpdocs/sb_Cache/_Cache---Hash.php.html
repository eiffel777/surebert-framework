Stores data in a hash. Data is only alive until the end of script, then all data is gone. You can still set lifetime, etc in case you script uses sleep has a long timeout

  1.  App::$cache = new sb_Cache_Hash(');
  2.  App::$cache->store('/users/paul', $user);
