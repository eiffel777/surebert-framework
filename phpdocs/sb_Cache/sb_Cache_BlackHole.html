Class sb_Cache_BlackHole

Description
Description | Methods (details)

Implements interfaces:

  • sb_Cache_Base

Does not actually cache any data, but uses the Cache interface so that you can

removed caching without changing application code

  1.  $cache = new sb_Cache_BlackHole();
  2.  $cache->store('/my/key', 'something');
  3.  echo $cache->fetch('/my/key');
